# AI Projects Repository

A collection of AI-powered and full-stack web applications developed using modern technologies like React.js, Node.js, Express.js, MongoDB, and Gemini AI.

This repository demonstrates frontend development, backend API integration, authentication systems, real-time communication, AI-powered features, and scalable MERN stack application architecture.
